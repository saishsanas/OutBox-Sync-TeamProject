# OutBox-Sync

A Spring Boot implementation of the Transactional Outbox Pattern for reliable event processing and failure handling.

## Technologies Used

* Java
* Spring Boot
* JPA / Hibernate
* MySQL
* Maven

## Features

* Transactional Outbox Pattern
* Retry Mechanism
* Event Processing
* Failure Handling
* REST APIs

## Contribution

Collaborative backend project involving development support, API testing, debugging, backend integration, and validation of event-processing workflows.
